## Tech Stack

- NodeJS
- Express
- JWT (Authentication)
- MySQL (Database)
- Prisma
- Yarn
- ReactJS

## Run Locally

- clone project

```Backend

1) npm install

2) npx prisma migrate dev

3) Start the server --> npm start

```

```Admin - CMS

1) npm install / yarn install


2) Start the server --> yarn dev

```

```BanHang - Shop

1) npm install / yarn install


2) Start the server --> yarn dev

```
